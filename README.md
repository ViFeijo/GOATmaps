<h1>APIs utilizadas no projeto</h1>

  <ul>
    <li><strong>Open-Meteo:</strong> dados de clima</li>
    <li><strong>Nominatim (OpenStreetMap):</strong> busca de locais e coordenadas</li>
    <li><strong>REST Countries:</strong> informações sobre paises</li>
    <li><strong>GeoJSON (GitHub/johan):</strong> dados geograficos</li>
    <li><strong>Leaflet:</strong> exibicao de mapas</li>
    <li><strong>ExchangeRate-API:</strong> taxas de cambio</li>
    <li><strong>GNews:</strong> noticias</li>
  </ul>
